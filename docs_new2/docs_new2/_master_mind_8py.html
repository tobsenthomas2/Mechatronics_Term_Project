Detailed Description

This file contains a demonstration program that checks the specific states of the gun.

This is the hub for all of our motor positioning/firing, and mainly interfaces with our thermal camera to get positioning and set the position and fire if needed. This sets four states, 0, 1, 2, and 3. State 0 initializes. It also hardcodes in the KP and position. State 1 waits for data from the other code. State 2 gets positional data. State 3 fires the gun. State 4 is an error state.
