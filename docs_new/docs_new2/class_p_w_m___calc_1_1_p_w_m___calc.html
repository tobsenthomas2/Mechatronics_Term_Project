Detailed Description

Class PWM_Calc: A class to implement a Proportional-Windup Controller.

The class stores the time, position, error, and PWM values and has functions to set the controller's parameters, run the control loop, and print the stored data to the serial port.

The constructor initializes the instance variables KP_set, Theta_Set, time, position, error, and pwm to 0.
